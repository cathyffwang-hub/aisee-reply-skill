💹 如何使用仪表盘1. 功能简介        仪表盘是一个数据可视化容器，让您能够轻松地将各种图表添加到智能表格中，从而实现对数据的多维度分析和呈现。通过仪表盘，您可以快速创建精美的业务图表，实时跟踪关键数据指标，从不同角度洞察数据变化，提升企业决策效率和质量。2. 操作流程step1：创建仪表盘 创建仪表盘：打开一份智能表格，点击左下角加号键，选择「新建仪表盘」（注：5月14日后新建智能表格，下方会自动出现「仪表盘」工作表），点击「添加图表」，选择「图表类型」，配置样式，选择一个数据表作为数据源，定义图表数据对应的具体字段，即可开始创作之旅。查看仪表盘：点击左上角「筛选」，添加筛选条件，即可得到定制化看板发送仪表盘：点击左上角「设置自动化发送」，可设定在添加新纪录/记录内容修改/定时重复/到达记录中时间时向指定文档用户推送仪表盘，仪表盘将以图片形式展示step2：选择不同仪表盘图形 👀 在仪表盘中，图表的横轴通常代表需要观察的维度，例如时间、地区、产品等，而纵轴则代表需要观察的指标，例如销售额、用户数量、转化率等。📈 根据不同的业务需求和数据特性，可以选择不同类型的图表进行展示，例如柱状图、条形图、折线图、散点图、饼状图等。每种图表都有其特定的应用场景和表达方式。👍 使用图表，您可以直观地了解数据的概览情况，从而做出更加精确和有效的决策。同时，图表也是会议汇报中的重要工具，可以帮助您更清晰、更生动地向他人展示数据分析的结果。|  |  |  |  |  |

2.0 sheet 为什么用不了跨文件引用函数呢？     由于跨文件引用函数技术架构设计问题，目前是只能在 3.0 sheet 上进行使用的。存量用户 2.0 sheet暂不能体验。跨表引用的数据会自动同步源数据吗？     当你跨表引用的源表(被引用的表格)数据发生变更时候，那么IMPORTRANGE公式所在表格将会自动同步源数据的变更，保证你能随时查看最新数据信息。我可以直接修改引用后的数据吗？     不可以，如果你在IMPORTRANGE公式引用数据成功后，想直接修改这部分跨表引用的数据是不允许的，会报错 #SPILL!。跨文件引用能引用智能表格吗？     跨文件引用功能不能引用智能表格数据。如何区分3.0 还是 2.0 表格？      可以让用户在表格输入"="，如果可以看到 IMPORTRANGE函数的提示，说明是3.0 表格。哪些表格文档可以被我引用呢？这里就涉及权限安全相关问题，首先你需要具备被引用文档的大表权限的“可复制权限”。1.如果你引用的是自己创建的表格，那么无论表格本身是什么权限（私密或者指定协作人可查看），你都是有权限引用的。     2.如果你引用的是别人创建的表格。   2.1 如果该表格是私密文档（仅我可查看），不管开不开启高级权限中的复制权限都是不能引用的。   2.2 如果该表格是指定协作人权限，先看是否在高级权限中开启了禁止复制（禁止查看复制、禁止编辑复制），如果开启了，那么是没有引用权限的，如果没有开启，那么看你是否在协作人列表，如果没有在协作人列表中，也是没有引用权限的。   2.3 如果该表格是所有人可查看权限，先看是否在高级权限中开启了禁止复制（禁止查看复制、禁止编辑复制），如果开启了，那么是没有引用权限的，如果没有开启，那么是有引用权限的。   2.4 如果该表格是所有人可编辑权限，先看是否在高级权限中开启了禁止复制（禁止查看复制、禁止编辑复制），如果开启了，那么是没有引用权限的，如果没有开启，那么是有引用权限的。当你具备被引用文档的大表权限后，还需要看你是否具备具体引用格子的区域权限。     1.如果你没有设置区域权限，那么肯定是可以引用的。     2.如果设置了区域权限。【这里待确认下，目前有点出入】2.1 仅我可编辑。2.2 指定人可查看、可编辑。如果报错提示没有引用表格的权限，怎么办呢？一般有以下两种情况会提示用户没有引用表格的权限。      1.用户写IMPPORTRANGE时候没有文档的引用权限；      2.用户开始时候有引用权限，后续权限被回收了。      用户只需要按照报错提示，有按钮点击，跳转到权限申请页面，进行权限申请即可，权限审批通过，即可自动正常引用。🗂如何使用智能分列1. 功能简介
